# Certificate

- Go to certificate menu showing certificate list page. There is default showing 10 rows.

<img src="../../admin/certificate-list.png" />

- When we need to add certificate click Add certificate button. Now showing Add certificate page.

- In the certificate page we need to add students for who want blockchain powered digital certificate.

- This students list adds to two way. The ways are, I.Individual, II.Bulk

### Individual

- Here have student basic details.

- We add multiple students (MAX:1000).

- The student details added lists are showing same page. There is anything need to particular student data we use update button.

- Once lists are ready click submit button.

### Bulk

- Click Bulk radio button.

- Follow the steps shown here.

- Once lists are ready click submit button.

<img src="../../admin/certificate-individual1.png" />

- Enter valid certificate student details and click submit button.
- Once students are created successfully return to certificate page. This created students showing in the list.

<img src="../../admin/certificate-list1.png" />

- Anything actions needed in the student list use action column edit icon and delete icon.

- Edit icon use to update the certificate student details.

- Delete icon used to delete this student.

- Once student lists are ready select current list dropdown new value.

- Select student list left side checkbox or check all checkbox.

<img src="../../admin/certificate-selected.png" />

- Once students selected click submit for blockchain certification button.

- Students data submitted blockchain can’t edit or delete.

- This selected max students are 100.

- This student list send their organisation mail.

- This student Count added in notification page.

<img src="../../admin/notification.png" />

- Few min back or within one day we submitted data are minted in block chain.

- The certificate minted blockchain view eye icon enabled.

<img src="../../admin/certificate-eye.png" />

- The view icon enabled click. Now showing certificate page.

- This certificate has student certificate, student all details and blockchain minted details.

- This certificate page have download option, view credentials and social share.

<img src="../../admin/credentials-validation.png" />
<img src="../../admin/certificate-page.png" />
