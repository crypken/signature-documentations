# Reset Password

> Click Forgot password.

<img src="../../admin/login.png" />

> Enter already registered email and click RESET PASSWORD button.

<img src="../../admin/reset-password.png" />

> Reset Password Success Page.

<img src="../../admin/reset-password-success.png" />

> Go to your mail id and click Reset password button.
> Enter password and confirm password then click submit button.

<img src="../../admin/reset-password-screen.png" />

> Reset password success return to login page and use this password to login.

> Reset Password Failure Page.

<img src="../../admin/reset-password-failure.png" />
